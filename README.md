# Rettory

Rendering Mod

Advanced rendering mod for Minecraft 1.20.1 compatible with Embeddium

## Installation
1. Download Forge 1.20.1
2. Download the latest release from releases
3. Put the JAR file in your mods folder
4. Launch Minecraft with Forge

## Building From Source
1. Install Java 17
2. Clone this repository
3. Run `gradlew.bat build` (Windows) or `./gradlew build` (Linux/Mac)
4. JAR file will be in `build/libs/`
